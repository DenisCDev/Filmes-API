# Filmes API
API que permite registro de filmes

## Tecnologias utilizadas
 - Java
 - Maven
 - Springboot
 - MongoDB

## Sobre os filmes
Cada filme possui diversos atributos pensandos para a melhor organização e cadastro, são os seguintes:
 - Titulo
 - Data de lançamento;
 - Sinopse;
 - Diretor;
 - Produtor;
 - Categoria;
 - Classificação;
 - Idiomas;
 - Nota;
 - Duração
